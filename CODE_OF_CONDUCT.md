# Código de Conduta

Este projeto adota o Contributor Covenant v2.1.

## Nosso compromisso
Prometemos tornar a participação em nossa comunidade uma experiência livre de assédio para todos.

## Nossos padrões
Comportamentos que contribuem para um ambiente positivo incluem:
- Linguagem acolhedora e inclusiva
- Respeito a diferentes pontos de vista
- Aceitar críticas construtivas

Comportamentos inaceitáveis incluem:
- Assédio, intimidação ou discriminação
- Comentários ofensivos

## Aplicação
Casos de abuso podem ser reportados via SECURITY.md.

## Atribuição
Este Código de Conduta é adaptado do Contributor Covenant, versão 2.1.
