# Contribuindo

Obrigado por contribuir com o InstaBytes!

## Como começar
1. Faça o fork do repositório
2. Crie uma branch: `git checkout -b feat/minha-feature`
3. Instale dependências: `npm install`
4. Rode os checks locais: `npm run lint`, `npm run test`, `npm run build`

## Padrões
- Use TypeScript e mantenha tipos atualizados.
- Prefira componentes pequenos e reutilizáveis.
- Evite efeitos colaterais fora de hooks.

## Commit
- Escreva mensagens claras e objetivas.

## Pull Request
- Descreva o contexto e impacto da mudança.
- Inclua testes quando aplicável.
