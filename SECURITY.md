# Política de Segurança

## Reportando Vulnerabilidades
Se você encontrar uma vulnerabilidade, por favor reporte de forma responsável.

1. Abra uma issue privada (se disponível) ou envie um e-mail para **security@exemplo.com**.
2. Descreva claramente o impacto, passos para reproduzir e possíveis mitigações.

## SLA
Vamos responder em até 7 dias úteis com um plano de ação.
